# admin-platform
一个界面优美，功能齐全的后台管理平台

## 演示地址

https://pmg1989.github.io/admin-platform

## 快速开始

克隆项目文件:

```
git clone https://github.com/pmg1989/admin-platform.git
```

进入目录安装依赖:

```
npm install 或者 yarn 或者 yarn install
```

开发：

```bash
run npm start or yarn start or gulp connect

打开 http://localhost:8888
```
